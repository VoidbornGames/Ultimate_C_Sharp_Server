# Security Policy for UltimateServer


## Supported UltimateServer Versions

| Version | Supported          |
| ------- | ------------------ |
| 7.5.x   | :white_check_mark: |
| 6.5.x   | :white_check_mark: |
| < 6.5.x | :x:                |


## Supported Ubuntu Versions

|  Version  |     Supported      |
| --------- | ------------------ |
|   24.04   | :white_check_mark: |
|   22.04   | :white_check_mark: |
| < 22.04   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in UltimateServer, please **do not post it publicly**.  
Instead, report it privately to the maintainers:

- Email: voidborn.games.official@gmail.com
- Or use GitHub's private Security Advisories feature: [GitHub Security Advisories](https://docs.github.com/en/code-security/security-advisories)

## What to Include

When reporting, please include:

- A detailed description of the vulnerability  
- Steps to reproduce it  
- Any affected versions or systems  
- Potential impact or severity

## Scope

This policy covers:

- The UltimateServer C# server code  
- Dependencies bundled with UltimateServer  
- Configuration files and deployment scripts

## Response

- Maintainers will acknowledge your report promptly.  
- We will work to fix the issue and release a patch before public disclosure.  
- Your contribution will be credited if desired.

## Responsible Disclosure

By reporting security issues privately, you help us protect the community and improve UltimateServer safely.
